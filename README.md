# 🏠 退去費用セカンドオピニオンAI (Taikyo Second Opinion)

賃貸物件の退去時に請求される「原状回復費用」が、国土交通省のガイドラインに沿った妥当なものかを客観的にチェックし、アドバイスを提供するWebアプリケーション（PoC版）です。

## 💡 プロジェクトの目的
退去費用をめぐるトラブルは、入居者と管理会社（大家）との「知識の非対称性」によって発生します。本アプリは、国土交通省が定める「原状回復をめぐるトラブルとガイドライン」という明確な基準と、ユーザーの退去書類をAI（LLM）を用いて比較し、不当な支払いを未然に防ぐための「気付き」を与えることを目的としています。

> **⚠️ 【重要】免責事項および非弁行為の回避について**
> 本アプリは、あくまで「ガイドラインや契約書の内容との差分を客観的事実として提示する」学習・サポートツールです。
> 弁護士法第72条に抵触する恐れがあるため、AIによる法的な判断や断定（「この請求は違法です」「支払う必要はありません」等）は一切行いません。最終的な交渉・判断はユーザーご自身の責任で行ってください。

---

## ✨ 主な機能 (MVPスコープ)
本PoCでは、トラブルが最も多い以下の「3つの重点項目」に絞ってAIがチェックを行います。
1. **壁紙（クロス）の張替え** （入居期間による減価償却の有無、張替え範囲）
2. **ハウスクリーニング代・鍵交換代** （原則貸主負担の確認、特約の有無）
3. **エアコン内部洗浄代** （原則貸主負担の確認、特約の有無）

### ユーザーインターフェース (Streamlit)
- **入居期間の入力**: 年・ヶ月をフォームから入力。
- **PDFアップロード機能**: 手入力を省くため、「退去費用見積書」および「賃貸借契約書（特約事項）」をPDFファイル形式でアップロード可能。
- **AIモデル選択**: PoC検証用として、サイドバー等から使用するGeminiモデル（例: `gemini-1.5-flash`, `gemini-1.5-pro`）を動的に切り替え可能。
- **結果の構造化表示**: AIから返却されたJSONデータをパースし、項目ごとの評価とアドバイスを画面上に分かりやすく整形して表示。

## 🛠️ 技術スタック
- **Frontend / Backend**: [Streamlit](https://streamlit.io/) (Python)
- **LLM API**: Google Gemini API (`gemini-1.5-pro`, `gemini-1.5-flash` 等)
- **PDF処理**: Gemini APIのネイティブドキュメント解析機能
- **環境変数管理**: `python-dotenv`

## 🚀 ローカル環境でのセットアップ手順

### 1. リポジトリのクローン
```bash
git clone https://github.com/yourusername/taikyo-second-opinion.git
cd taikyo-ai-second-opinion
```

### 2. 必要なライブラリのインストール
Python 3.10以上を推奨します。仮想環境（venv等）を作成の上、以下のコマンドを実行してください。
```bash
pip install -r requirements.txt
```

### 3. 環境変数の設定
プロジェクトのルートディレクトリに `.env` ファイルを作成し、Google AI Studioで取得したGemini APIキーを設定してください。
```env
GEMINI_API_KEY="your_gemini_api_key_here"
```

### 4. アプリケーションの起動
```bash
streamlit run app.py
```
ブラウザが自動的に開き、`http://localhost:8501` でアプリが起動します。

## 📝 使い方 (PoC版)
1. 画面の指示に従い、**「入居期間」** を入力します。
2. 管理会社から受け取った **「退去費用見積書」** のPDFをアップロードします。
3. 手元にあれば **「賃貸借契約書（特約事項が記載されたページ）」** のPDFをアップロードします（任意）。
4. サイドバーから検証したい **Geminiモデル** を選択します。
5. 「チェック開始」ボタンを押すと、AIがガイドラインと照らし合わせた結果を画面に表示します。
